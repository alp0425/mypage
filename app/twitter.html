Twitter Mood Monitor

A Twitter sentiment visualization tool

Twitter mood monitor is a graphic user interface which interprets the affective content from Twitter and allows user to access such information via visual representations. I would like to pay tribut to my advisor Emmanuel G. Blanchard for giving me great advice during the process.

Motivation

I wish that I could say that I wanted to solve a real problem from this project. But my motivation was purely curiosity about people. I wanted to know more about people’s perceptions about different things. Then I thought about Twitter, it is the one social network that all about perceptions, opinions and feelings. It was started from a private fun project. It went further until it became my master thesis project.

Concept

I started from researching the avaliable Twitter sentiment analysis tools online. I quickly found out the majority of those tools only provide a binary answers, meaning the result is either good or bad. I in I wanted to give more sentiment dimensions and give more space for post analysis to my users.

System design

I applied the k-mean classification to find the coordinate of each tweets on the canvas. But I was still not satisfied with the space of post interpretation that my users will have. Therefore, I also calculated the accuracy of each measurement to give my user an even richer picture.

Interface Design

User stated by searching an interesting topic in keywords. The retrieved tweets will then grouped and displayed on the sentiment canvas. The sentiment canvas was redesigned from the Russell dimensional model.

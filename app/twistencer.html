Twistencer

The music installation which brings people together

The Roskilde Festival was hosting an innovation project during 2010-2011. The objective was to enhancing innovation of Denmark. Our project “Twistencer” was luckily chosen as one of the 8 innovation projects. We finished this project in 4 months from the concept design to the 3 working prototypes. We were having quite a few crazy deadlines, but I had great fun for making this project.

The social icebreaker

Twistenser is an interactive sound installation that was inspired by Twister (the board game) and sequencer (The electronic musical instrument). Music and board games are both great icebreaker at social event. Therefore we believe the Twistenser will bring people together on the Roskilde Festival.

Sensor Design

I was taking charge of sensor design and implementation. Twistenser requires pressure sensor which send electric signal where user interact with it. The main obstacle that I was facing was the sensor load bearing issue. They have to be able to support human weight and still remain good conductivity.

Prototypes

We built this instillation from the scratch. It was made from wood board, aluminum plate, plastics and sponges. We used the arduino board to control the sensors and music sequence. We finished this project in 4 months from the concept design to the 3 working prototypes. We were having quite a few crazy deadlines, but I had great fun for making this project :)

About

Quizpedia is a mobile application for people to create and share quiz with the others. My role in the project was to provide responsive designs and build the front-end using HTML, CSS, and CoffeeScript. A big part of my work required collaborations with designers and other developers.

What makes quizpedia different from the other quiz games is the customizable feature. It allows user to create all kinds of quiz with not only text, but also embedded videos and sound. Therefore, we wanted to highlight this feature on the website with simple interactions to show how easy it is to create an amazing quiz.

Journey Takeaways

As one of the game fanatics, it was a great opportunity for me to work with quizpedia team. I have learned so much from the senior developers. They were giving me a lot of instructions about how to structure my code, which will benefit for the future development.
